HTML stands for Hypertext Markup Language

Hypertext: pieces of text which contain link to other text thus connecting webpages to one another, either within a single website or between websites.

Markup Language: bold, italic, underlin... HTML uses Tags  to markup/anonate texts, images or contents like- <h1>, <p>, <b>, etc

1. HTML Heading Elements [13]:

<h1> Hello World </h1>

<h1> is opening tag.
Hello World is the content of the tag.
</h1> is closing tag contains forward slash.

Anything between <> is called "Tag".
while the entire "<h1> Hello World </h1>" is called "Element".

2. HTML Paragraph Elements [14]:

Adds a line between two paragraph elements.

Code:
<p> First Paragraph </p>
<p> Second Paragraph </p>
<p> Third Paragraph </p>

Output:
First Paragraph

Second Paragraph

Third Paragraph


3. Self Closing Tags [15]:
Void Elements, elements with no contents between opening and closing tags. Infact the tags itself is different: <hr /> (Horizontal rule element).
<p> First Paragraph </p>
<hr />
<p> Second Paragraph </p>

Output:
First Paragraph

-------------------------------------------------------------------------

Second Paragraph

Break Element <br />:
Poems for example contains different lines all in same paragraph in order for rhyming to work. <br /> is used to add new line.

<p>
To see a World in a Grain of Sand <br /> And a Heaven in a Wild Flower,<br /> Hold Infinity in the palm of your hand<br /> And Eternity in an hour.
<p>

Output:

To see a World in a Grain of Sand
And a Heaven in a Wild Flower,
Hold Infinity in the palm of your hand
And Eternity in an hour.

Exersice:
<h1>Angela's Cinnamon Roll Recipe</h1>

<h2>Ingredients</h2>

<h3>For the dough:</h3>
<ul>
<li>¾ cup warm milk</li>
<li>2 ¼ teaspoons yeast</li> 
<li>¼ cup granulated sugar</li>
<li>1 egg plus 1 egg yolk</li>
<li>¼ cup butter</li>
<li>3 cups bread flour</li>
</ul>

<h3>For the filling:</h3>
<ul>
<li>2/3 cup dark brown sugar</li> 
<li>1 ½ tablespoons ground cinnamon</li>
<li>¼ cup butter</li>
</ul>

<h2>Instructions</h2>
<ol>
<li>Mix the milk with the yeast, sugar, eggs.</li>
<li>Melt the butter and add to the mixture.</li>
<li>Add in the flour and mix until combined into a dough.</li>
<li>Knead the dough for 10 minuites. </li>
<li>Transfer the dough into a large bowl and cover with plastic wrap. Leave it somewhere to rise for 2 hours.</li>
<li>After the dough has doubled in size, roll it out into a large rectangle. </li>
<li>Melt the butter for the filling and mix in the sugar and cinnamon.</li>
<li>Spread the filling onto the dough then roll the dough into a swiss roll.</li> 
<li>Cut the roll into 3cm sections and place flat into a baking tray.</li>
<li>Pre-heat the oven to 350F or 180C, then bake the rolls for 20-25min until lightly brown.</li>
</ol>

Attributes of <ol>

value

    This integer attribute indicates the current ordinal value of the list item as defined by the <ol> element. The only allowed value for this attribute is a number, even if the list is displayed with Roman numerals or letters. List items that follow this one continue numbering from the value set. This attribute has no meaning for unordered lists (<ul>) or for menus (<menu>).

start
An integer to start counting from for the list items. Always an Arabic numeral (1, 2, 3, etc.), even when the numbering type is letters or Roman numerals. For example, to start numbering elements from the letter "d" or the Roman numeral "iv," use start="4".

type

    Sets the numbering type:

        a for lowercase letters
        A for uppercase letters
        i for lowercase Roman numerals
        I for uppercase Roman numerals
        1 for numbers (default)

9. HTML Boiler Plate [27]

<!DOCTYPE html> //Tells the browser which version of HTML
<html lang="en"> //root of the document. Setting language to english (useful for screen reader)
  <head>
    <meta charset="UTF-8">
    <title>My Website</title>//gets displayed in the tab menu
  </head>

  <body>
    <h1>Hello World!</h1>
  </body>
</html>



Inside head importatnt information about the website it placed. Nothing here is visible to users.
It helps website render in the browser correctly. Doesnt include text or images.
